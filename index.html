CSS Snippet Cheatsheet

This page is a collection of common CSS code that you can easily grab and copy to add to your projects!

Rounded Fancy Box

Creates a simple rounded corner fancy box.


            .box {
              border-radius: 25px;
              border-style: solid;
              border-width: 1px;
              padding: 10px;
              margin: 20px;
            }

Box Transition Glow

Use transition and box shadows to glow on hover.


            .card .card-header {
              border-radius: 7px;
              transition: all 0.5s ease-in-out;
            }
            
            .card:hover,
            .card:hover .card-header {
              box-shadow: inset 0px 0px 8px #fff
            }

Media Query for Mobile

Media Query example for mobile responsive.


            @media screen and (max-width: 768px) {
              .class-selector {
                property1: value1;
                property2: value2;
              }
            
              element-selector {
                property1: value1;
              }
            }

Flexbox Navbar

Example of a flexbox navbar.


            .nav {
              display: flex;
              justify-content: flex-end;
            }

Flexbox Row

Use to create a Flexbox row layout.


            .row {
              display: flex;
              flex-direction: row;
              flex-wrap: wrap;
            }

Flexbox Column

Use to create a Flexbox column layout,


            .column {
              display: flex;
              flex-direction: column;
            }
