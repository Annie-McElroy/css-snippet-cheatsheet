CSS Snippet Cheatsheet

This page is a collection of common CSS code that you can easily grab and copy to add to your projects!

Rounded Fancy Box

Creates a simple rounded corner fancy box.


            .box {
              border-radius: 25px;
              border-style: solid;
              border-width: 1px;
              padding: 10px;
              margin: 20px;
            }

Box Transition Glow

Use transition and box shadows to glow on hover.

Media Query for Mobile

Media Query example for mobile responsive.


            @media screen and (max-width: 768px) {
              .class-selector {
                property1: value1;
                property2: value2;
              }
            
              element-selector {
                property1: value1;
              }
            }
